# Unit 6: Array

## Overview
Up to this point, if you needed to store 50 test scores, you would have to create 50 separate variables. That's highly inefficient. In Unit 6, you will learn about **Arrays**, which allow you to store multiple values of the same type in a single, organized structure.

## Key Concepts to Research
1. **Creating and Initializing Arrays:** Learn the syntax to declare and create arrays of primitives and objects (e.g., `int[] scores = new int[50];`).
2. **Accessing Elements:** Understand how to use zero-based indexing to access and modify individual elements within an array.
3. **Array Length:** Learn how to use the `.length` property (notice there are no parentheses) to find the size of an array.
4. **Traversing Arrays:** How to use a standard `for` loop to iterate through every element in an array.
5. **The Enhanced `for` Loop (For-Each Loop):** Learn a simpler syntax for traversing an array when you don't need the index.
6. **Standard Algorithms:** Research common algorithms for arrays:
    *   Finding the minimum or maximum value.
    *   Computing a sum, average, or count.
    *   Checking if at least one or all elements have a certain property.
    *   Reversing an array.

## Essential Vocabulary
*   **Element:** A single value stored within an array.
*   **Index:** An integer value indicating the position of an element within an array (starting at 0).
*   **Array Bounds:** The valid indices for an array (from `0` to `length - 1`).
*   **ArrayIndexOutOfBoundsException:** An error thrown when your code tries to access an index that does not exist.

## Practice Ideas
*   Write a program that creates an array of 10 random integers and finds the largest value.
*   Write a program that takes an array of strings and prints only those strings that start with the letter "A".
