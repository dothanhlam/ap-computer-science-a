# Unit 2: Using Objects

## Overview
In Unit 1, you learned about primitive types, which hold single, simple values. In Unit 2, you will learn about **Objects**, which are more complex entities that can hold multiple pieces of data (attributes) and perform actions (methods). You will also learn about a special class: `String`.

## Key Concepts to Research
1. **Classes and Objects:** Understand the relationship between a Class (the blueprint) and an Object (the specific instance).
2. **Creating Objects:** Learn how to use the `new` keyword and constructors to instantiate objects.
3. **Methods and Parameters:** How to call methods on an object using dot notation (`object.method()`) and pass information (arguments/parameters) to them.
4. **Return Types:** Understanding the difference between methods that return a value (like `Math.abs()`) and methods that do not (`void` methods).
5. **The `String` Class:** Strings are objects, not primitives! Learn essential String methods: `length()`, `substring()`, `indexOf()`, `equals()`, and `compareTo()`.
6. **Wrapper Classes:** Learn about `Integer` and `Double`, and the concept of autoboxing and unboxing.
7. **The `Math` Class:** Familiarize yourself with `Math.random()`, `Math.abs()`, `Math.pow()`, and `Math.sqrt()`.

## Essential Vocabulary
*   **Constructor:** A special block of code used to initialize a newly created object.
*   **Instance:** A specific realization of any object.
*   **Null:** A special value indicating that an object reference does not point to any object.
*   **Overloading:** When a class has multiple constructors or methods with the same name but different parameter lists.

## Practice Ideas
*   Write a program that uses `String` methods to extract the first name and last name from a full name string.
*   Write a program that simulates a dice roll using `Math.random()`.

## Research Resources
*   Look up the Java API Documentation for `java.lang.String` and `java.lang.Math`.
*   Search for "Java passing parameters by value" to understand how arguments are handled.
