# Advanced Homework

- **String Manipulator:** Create an interactive program that asks the user for a sentence and performs various `String` methods on it (e.g., counting vowels, finding the index of specific characters, reversing substrings).
- **Math Game:** Build a random number generation game using the `Math` class where the user has to guess a generated number or solve a generated math problem involving `Math.pow` or `Math.sqrt`.
