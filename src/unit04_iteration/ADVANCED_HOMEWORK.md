# Advanced Homework

- **Mathematical Approximations:** Implement loops to calculate the Fibonacci sequence up to N terms or approximate the value of Pi using the Gregory-Leibniz series.
- **Reverse Guessing Game:** Create a game where the *user* thinks of a number between 1 and 100, and the *program* guesses it using a binary search approach and a `while` loop, asking the user if the guess is too high or too low.
