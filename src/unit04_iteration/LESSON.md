# Unit 4: Iteration

## Overview
What happens when you need to repeat a block of code 10 times, or 1,000 times? Writing the same lines over and over is inefficient and prone to errors. In Unit 4, you will learn about **iteration** (loops), which allows you to repeat a block of code as long as a certain condition remains true.

## Key Concepts to Research
1. **The `while` Loop:** Learn how to create loops that continue executing as long as a boolean condition evaluates to `true`.
2. **The `for` Loop:** Understand the three parts of a `for` loop header: initialization, condition, and update. Learn when to use a `for` loop instead of a `while` loop.
3. **String Traversals:** Learn how to use a loop along with `String.substring()` or `String.charAt()` to examine every character in a String.
4. **Nested Loops:** How to place a loop inside another loop. Understand how the inner loop completes all its iterations for every single iteration of the outer loop.
5. **Infinite Loops:** Learn what causes a loop to run forever and how to avoid it.
6. **Off-By-One Errors:** A common logical error where a loop executes one time too many or one time too few.

## Essential Vocabulary
*   **Iteration:** A single execution of the code block within a loop.
*   **Loop Control Variable:** A variable used to regulate the flow of control of the loop (e.g., the `i` in `for (int i = 0; i < 10; i++)`).
*   **Trace Table:** A technique used to track the values of variables as a loop executes, which is crucial for debugging.

## Practice Ideas
*   Write a program that uses a `while` loop to print all the even numbers between 1 and 100.
*   Write a program that reverses a String provided by the user using a `for` loop.
