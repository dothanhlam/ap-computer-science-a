# Unit 1: Primitive Types

## Overview
Welcome to Unit 1! In this unit, you will learn the foundational building blocks of Java programming. Before we can build complex programs, we need to understand how Java stores, names, and manipulates basic data.

## Key Concepts to Research
1. **The Anatomy of a Java Program:** Learn about the `main` method, classes, and how a basic Java program is structured.
2. **Variables and Data Types:** Understand the core primitive types in AP CS A: `int`, `double`, and `boolean`.
3. **Declaring and Initializing Variables:** Learn the syntax to create variables and assign them values (`int age = 16;`).
4. **Arithmetic Expressions:** Using operators (`+`, `-`, `*`, `/`, `%`). **Crucial:** Understand how integer division differs from double division.
5. **Compound Assignment Operators:** Shortcuts like `+=`, `-=`, `*=`, `/=`, `%=`, `++`, and `--`.
6. **Casting:** Explicitly converting between data types (e.g., `(int)` or `(double)`) and understanding data loss (truncation).

## Essential Vocabulary
*   **Variable:** A named memory location used to store a value.
*   **Primitive Type:** The most basic data types in Java, storing simple values directly.
*   **Modulo Operator (`%`):** Returns the remainder of a division operation (e.g., `10 % 3` is `1`).
*   **Truncation:** Dropping the decimal part when converting a double to an int (e.g., `3.99` becomes `3`).

## Practice Ideas
*   Write a program to calculate the average of three integers. Pay close attention to integer division.
*   Experiment with the modulo operator (`%`)—how can you use it to extract the last digit of an integer?
