# Unit 8: 2D Arrays

## Overview
What if you need to represent a grid, a game board like Tic-Tac-Toe, or a spreadsheet? A single 1D array isn't enough. In Unit 8, you will learn about **2D Arrays**, which are essentially arrays of arrays.

## Key Concepts to Research
1. **Declaring and Initializing 2D Arrays:** Learn the syntax to create a grid (e.g., `int[][] grid = new int[row][col];`).
2. **Accessing Elements:** Understand how to use row and column indices (`grid[r][c]`) to access specific cells.
3. **Dimensions:** Learn how to find the number of rows (`grid.length`) and the number of columns (`grid[0].length`).
4. **Row-Major Traversal:** How to use nested `for` loops to process a 2D array row by row.
5. **Column-Major Traversal:** How to use nested `for` loops to process a 2D array column by column.
6. **Enhanced `for` Loops with 2D Arrays:** Learn how to traverse a 2D array using an outer enhanced loop for rows (which are 1D arrays) and an inner enhanced loop for the elements.
7. **Algorithms on 2D Grids:** Learn how to apply standard algorithms (sum, min, max, searching) to 2D structures.

## Essential Vocabulary
*   **2D Array:** A data structure that stores values in a two-dimensional grid of rows and columns.
*   **Row-Major Order:** Iterating through all the elements of the first row before moving to the second row.
*   **Column-Major Order:** Iterating through all the elements of the first column before moving to the second column.

## Practice Ideas
*   Write a method that takes a 2D array of integers and calculates the sum of all elements.
*   Write a method that extracts a specific column from a 2D array and returns it as a 1D array.
